# 压缩与打包

## 压缩文件名与对应程序

| 扩展名 | 压缩程序 |
|:---|:---|
| *.Z | compress |
| *.zip | zip |
| *.gz | gzip |
| *.bz2 | bzip2 |
| *.xz | xz |
| *.tar | tar 打包（未压缩） |
| *.tar.gz | tar + gzip |
| *.tar.bz2 | tar + bzip2 |
| *.tar.xz | tar + xz |

## 压缩指令

### gzip

Linux 使用最广的压缩指令，可解 compress、zip、gzip 格式。压缩后源文件消失。

```bash
gzip [-cdtv#] filename
-c ：将压缩数据输出到屏幕
-d ：解压缩
-t ：检验压缩文件
-v ：显示压缩比
-# ：压缩等级 1-9，默认为 6
```

查看命令：`zcat`、`zmore`、`zless`

### bzip2

提供比 gzip 更高的压缩比。

```bash
bzip2 [-cdkzv#] filename
-k ：保留源文件
```

查看命令：`bzcat`、`bzmore`、`bzless`、`bzgrep`

### xz

提供比 bzip2 更佳的压缩比（但压缩时间也更长）。

```bash
xz [-dtlkc#] filename
```

查看命令：`xzcat`、`xzmore`、`xzless`、`xzgrep`

## 打包（tar）

tar 可将多个文件打包成一个大文件，并可调用 gzip/bzip2/xz 进行压缩。

```bash
tar [-z|-j|-J] [cv] [-f 新建的 tar 文件] filename...   # 打包压缩
tar [-z|-j|-J] [tv] [-f 已有的 tar 文件]               # 查看
tar [-z|-j|-J] [xv] [-f 已有的 tar 文件] [-C 目录]     # 解压缩

-z ：使用 gzip
-j ：使用 bzip2
-J ：使用 xz
-c ：新建打包文件
-t ：查看打包文件内容
-x ：解打包或解压缩
-v ：显示正在处理的文件名
-f ：指定要处理的文件
-C 目录：在指定目录解压缩
```

**使用示例**：

```bash
tar -jcv -f filename.tar.bz2 要压缩的文件或目录   # 打包压缩
tar -jtv -f filename.tar.bz2                      # 查看
tar -jxv -f filename.tar.bz2 -C 要解压缩的目录     # 解压缩
```

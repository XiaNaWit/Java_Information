# 网络管理

## 网络接口相关

| 命令 | 说明 |
|:---|:---|
| `ifconfig` | 查看/配置网络接口信息 |
| `ifup <接口>` | 开启网络接口 |
| `ifdown <接口>` | 关闭网络接口 |

## 临时配置

| 命令 | 说明 |
|:---|:---|
| `route` | 临时设置内核路由表 |
| `hostname` | 临时修改主机名 |
| `sysctl` | 临时开启内核包转发等参数 |
| `ifconfig` | 临时设置网络接口 IP 参数 |

## 网络检测工具

| 命令 | 说明 |
|:---|:---|
| `ping` | 检测网络连通性 |
| `netstat` | 查看网络状态、端口占用 |
| `traceroute` | 检测到目的主机所经过的路由器 |
| `tcpdump` | 显示本机网络流量状态 |
| `lsof` | 查看指定 IP/端口的进程 |
| `host` / `dig` / `nslookup` | 检测 DNS 解析 |

## 安装软件

| 命令 | 说明 |
|:---|:---|
| `yum` | RPM 包管理器（Red Hat / CentOS） |
| `rpm` | RPM 软件包管理 |
| `wget` | 网络下载工具 |

## 包管理工具

| 类型 | 包格式 | 包管理工具 | 发行版 |
|:---|:---|:---|:---|
| RPM 系 | .rpm | yum / dnf | Red Hat / Fedora / CentOS |
| DEB 系 | .deb | apt / dpkg | Debian / Ubuntu |

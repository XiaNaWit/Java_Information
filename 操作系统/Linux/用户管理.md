# 用户管理

## 用户管理

| 命令 | 说明 |
|:---|:---|
| `useradd` | 添加用户 |
| `usermod` | 修改用户 |
| `userdel` | 删除用户 |

## 组管理

| 命令 | 说明 |
|:---|:---|
| `groupadd` | 添加组 |
| `groupmod` | 修改组 |
| `groupdel` | 删除组 |

## 批量管理用户

| 命令 | 说明 |
|:---|:---|
| `newusers` | 成批添加/更新一组账户 |
| `chpasswd` | 成批更新用户口令 |

## 组成员管理

```bash
gpasswd -a <用户账号名> <组账号名>   # 向组添加用户
gpasswd -d <用户账号名> <组账号名>   # 从组删除用户
usermod -G <组账号名> <用户账号名>   # 设置用户附加组
```

## 口令管理

| 命令 | 说明 |
|:---|:---|
| `passwd` | 设置/修改用户口令 |
| `chage` | 设置已存在用户的口令时效 |
| 修改 `/etc/login.defs` | 配置口令时效默认参数 |

## 切换用户

| 命令 | 说明 |
|:---|:---|
| `su` | 切换用户 |
| `sudo` | 以 root 权限执行命令（需在 /etc/sudoers 中配置） |

## 用户相关命令

| 命令 | 说明 |
|:---|:---|
| `id` | 显示用户当前 uid、gid 和所属组列表 |
| `groups` | 显示指定用户所属组列表 |
| `whoami` | 显示当前用户名 |
| `w` / `who` | 显示登录用户及相关信息 |
| `newgrp` | 转换用户当前组到指定组 |

## 关机

```bash
who                       # 查看在线用户
sync                      # 同步磁盘数据
shutdown [-krhc] 时间 [信息]
  -k ：仅发警告不关机
  -r ：重启
  -h ：关机
  -c ：取消 shutdown
```

## Vi 编辑器

### 三种模式

- **一般指令模式**（Command mode）：默认模式，移动光标查看内容
- **编辑模式**（Insert mode）：按 `i` 等键进入，编辑文本
- **指令列模式**（Bottom-line mode）：按 `:` 进入，保存退出

### 一般模式常用键

| 按键 | 说明 |
|:---|:---|
| `G` | 跳转到文件尾 |
| `ZZ` | 存盘退出 |
| `ZQ` | 不存盘退出 |
| `/` 或 `?` | 查找字符串 |
| `n` | 继续查找下一个 |
| `yy` | 复制一行 |
| `p` | 粘贴在下一行 |
| `P` | 粘贴在前一行 |
| `dd` | 删除一行 |
| `u` | 撤销（Undo） |

### 指令列模式命令

| 命令 | 说明 |
|:---|:---|
| `:w` | 保存 |
| `:w newfile` | 另存为新文件 |
| `:q` | 退出（未修改时） |
| `:q!` | 强制退出不保存 |
| `:wq` | 保存并退出 |
| `:wq!` | 强制保存并退出 |

### 设置 Vi 环境

```bash
:set autoindent           # 缩进
:set noautoindent         # 取消缩进
:set number               # 显示行号
:set tabstop=value        # 设置制表符空格数
:set                      # 显示所有设置选项
```

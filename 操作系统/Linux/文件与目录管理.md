# 文件与目录管理

## 文件属性

使用 `ls -l` 查看文件信息，例如 `drwxr-xr-x 3 root root 17 May 6 00:14 .config`：

| 字段 | 说明 |
|:---|:---|
| drwxr-xr-x | 文件类型及权限，第 1 位为类型，后 9 位为权限 |
| 3 | 链接数 |
| root | 文件拥有者 |
| root | 所属群组 |
| 17 | 文件大小 |
| May 6 00:14 | 最后修改时间 |
| .config | 文件名 |

**文件类型**：d 目录、- 文件、l 链接文件。

**权限字段**（9 位，每 3 位一组）：r 读、w 写、x 执行。三组分别对应文件拥有者、所属群组、其他人。

**文件时间**：
- mtime（modification time）：文件内容更新时更新
- ctime（status time）：文件状态（权限、属性）更新时更新
- atime（access time）：读取文件时更新

## 文件与目录基本操作

### ls

列出文件或目录信息。

```bash
ls [-aAdfFhilnrRSt] file|dir
-a ：列出全部文件
-d ：仅列出目录本身
-l ：以长数据串行列出
```

### cd

切换当前目录。

```bash
cd [相对路径或绝对路径]
```

### pwd

显示当前工作目录。

### mkdir

创建目录。

```bash
mkdir [-mp] 目录名称
-m ：配置目录权限
-p ：递归创建目录
```

### rmdir

删除空目录。

```bash
rmdir [-p] 目录名称
-p ：递归删除空目录
```

### touch

更新文件时间或创建新文件。

```bash
touch [-acdmt] filename
-a ：更新 atime
-c ：更新 ctime，文件不存在时不创建
-m ：更新 mtime
-d ：指定更新日期
-t ：指定更新时间，格式 [YYYYMMDDhhmm]
```

### cp

复制文件或目录。

```bash
cp [-adfilprsu] source destination
-a ：相当于 -dr --preserve=all
-d ：若来源为链接文件，则复制链接文件属性
-i ：目标文件存在时覆盖前询问
-p ：连同文件属性一起复制
-r ：递归复制
-u ：目标比源旧才更新
```

### mv

移动文件或目录，也可用于改名。

```bash
mv [-fiu] source destination
-f ：强制覆盖
```

### rm

删除文件或目录。

```bash
rm [-fir] 文件或目录
-r ：递归删除
```

### echo

将内容输出到指定文件，有则打开无则创建。

### file / stat

查看文件类型或文件属性信息。

### 重定向

```bash
echo "内容" > file.txt    # 覆盖模式
echo "内容" >> file.txt   # 追加模式
```

## 修改权限

### 数字法

每组权限的 3 个位做二进制数字，权值为 r:4、w:2、x:1。

```bash
chmod [-R] xyz dirname/filename
```

示例：`chmod 754 .bashrc` 表示 `-rwxr-xr--`。

### 符号法

```bash
chmod [ugoa] [+-=] [rwx] dirname/filename
u：拥有者  g：所属群组  o：其他人  a：所有人
+：添加权限  -：移除权限  =：设定权限
```

示例：`chmod a+w .bashrc` 为所有用户添加写权限。

### 默认权限 umask

| 类型 | 默认权限 |
|:---|:---|
| 文件 | 666（-rw-rw-rw-），无可执行权限 |
| 目录 | 777（drwxrwxrwx），必须有可执行权限 |

`umask 002` 表示其他用户去除写权限，新文件默认 `-rw-rw-r--`。

## 目录的权限

文件名存储在目录中。目录的 r 权限可读取文件列表；w 权限可修改文件列表（增删改名）；x 权限使目录成为工作目录，是 r 和 w 的基础。

## 链接

```bash
ln [-sf] source_filename dist_filename
-s ：符号链接（默认是硬链接）
-f ：目标存在时先删除
```

### 硬链接

在目录下创建条目，记录文件名与 inode 编号，与源文件共用同一个 inode。删除任意条目文件仍存在（引用数不为 0）。限制：不能跨文件系统，不能对目录链接。

### 符号链接

保存源文件的绝对路径，类似 Windows 快捷方式。源文件删除后链接打不开。可以为目录建立符号链接。

## 获取文件内容

| 命令 | 说明 |
|:---|:---|
| cat [-AbEnTv] | 获取文件内容，-n 显示行号 |
| tac | 从最后一行开始打印 |
| more | 分页查看，适合大文件 |
| less | 类似 more，支持向前翻页和搜索 |
| head [-n number] | 取前几行 |
| tail [-n number] | 取后几行 |
| od | 以字符或十六进制显示二进制文件 |

## 指令与文件搜索

| 命令 | 说明 |
|:---|:---|
| which [-a] command | 指令搜索 |
| whereis [-bmsu] | 文件搜索，只搜索特定目录，速度快 |
| locate [-ir] keyword | 文件搜索，支持正则，基于数据库（/var/lib/mlocate/） |
| find [basedir] [option] | 文件搜索，可用属性和权限搜索 |

### find 常用选项

**时间相关**：`-mtime n`（n 天前）、`-mtime +n`（n 天之前）、`-mtime -n`（n 天之内）、`-newer file`（比 file 新）

**拥有者**：`-uid n`、`-gid n`、`-user name`、`-group name`、`-nouser`、`-nogroup`

**名称与权限**：`-name filename`、`-size [+-]SIZE`、`-type TYPE`、`-perm mode`、`-perm -mode`、`-perm /mode`

## 文件扩展属性

```bash
lsattr [-adR] [文件|目录]          # 显示扩展属性
chattr [-R] [[-+=][属性]] <文件|目录>  # 修改扩展属性
```

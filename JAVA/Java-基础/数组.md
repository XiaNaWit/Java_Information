# 数组

## 初始化

### 静态初始化

```java
int[] a = {1, 2, 3};
int[] b = new int[]{1, 2, 3};
```

### 动态初始化

```java
int[] a = new int[3];  // 默认值 0
```

## 多维数组

```java
int[][] matrix = new int[3][4];                  // 3行4列
int[][] arr = {{1, 2}, {3, 4}, {5, 6}};
```

## Java 数组和内存

数组引用存储在栈中，数组元素存储在堆中。

```java
int[] array = new int[]{1, 2, 3};
// array（栈） → {1,2,3}（堆）
```

## Arrays 工具类

```java
int[] a = {3, 1, 4, 1, 5, 9};

Arrays.sort(a);                          // 排序
int idx = Arrays.binarySearch(a, 4);     // 二分查找（需先排序）
int[] copy = Arrays.copyOf(a, 10);       // 拷贝（指定新长度）
int[] range = Arrays.copyOfRange(a, 1, 4); // 拷贝范围
Arrays.fill(a, 0);                       // 填充
String str = Arrays.toString(a);         // 转字符串
boolean eq = Arrays.equals(a, b);        // 判断相等
```
